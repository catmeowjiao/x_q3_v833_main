## 怎么从项目源代码编译
* 建议使用**linux系统** 
* 克隆项目到一个空文件夹中  
* 安装交叉编译器
> 例如armhf musl gcc编译器
* 将项目文件中build.sh的文件路径改为你的编译器路径<br>
* 运行 ` ./build.sh `
* 目标文件为"demo"
